Readme - LonTalkStack Simple Device Example

DESCRIPTION:	
 The LonTalkStack simple device example implements a simple LonTalkStack device 
 that supports a single input and single output network variable.  See the 
 comments at the top of main.c for more information.
 
 Note that this example code is shared by both the SimpleIp852Device and the 
 SimpleLtDevice Visual Studio projects.
